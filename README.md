# JSP with Servlet CRUD Project

## Description
This project is a **web application** built using **JSP and Servlets** to perform CRUD (Create, Read, Update, Delete) operations. It demonstrates how to build a simple Java web application that interacts with a MySQL database using Servlets as the backend and JSP for the frontend.

## Features
- User management with full CRUD operations.
- Frontend built with JSP.
- Backend logic handled with Java Servlets.
- MySQL database integration.
- Easily extendable for larger applications.

## Technologies Used
- Java 8+
- JSP & Servlet
- Apache Tomcat
- MySQL
- HTML & CSS

## Setup & Installation
1. Clone the repository:
```bash
git clone https://github.com/yourusername/jsp-servlet-crud.git
Import the project into your IDE (Eclipse or IntelliJ IDEA).

Ensure Apache Tomcat is configured.

Update database credentials in the configuration file.

Deploy the project on Tomcat.

Running the Project
Start Apache Tomcat server.

Open your browser and navigate to http://localhost:8080/[your-app-name].

Perform CRUD operations via the web interface.

- Full CRUD operations for managing users.
- RESTful APIs for backend operations.
- MySQL database integration.

- Easily scalable for larger applications.

- Java 8+
- Spring Boot

- HTML & Thymeleaf (optional frontend)
